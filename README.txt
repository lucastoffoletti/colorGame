COLORGAME se trata de un juego basado en una paleta de colores(RGB red, green, blue) donde cada color tiene un porcentaje númerico y el objetivo del juego es adivinar en base a dicho porcentaje, de que color se trata. El juego cuenta con un nivel EASY MODE y un nivel HARD MODE. 
Cuando la elección del jugador haya sido la correcta, se pintaran todos los cuadrantes del color elegido y se lanzara un texto con la palabra "Correcto!" En caso contrario, tendrá la posibilidad de seguir intentandolo nuevamente.
